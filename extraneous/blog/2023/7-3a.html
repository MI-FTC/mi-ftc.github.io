It's been another full day of work on the site, and there's quite a bit of new stuff today. I'm just going to list it off:

  • Wrote the first blog article (you're reading it right now!) - it's under the "extraneous" folder because it's not really that useful but it might serve to explain some things on occasion (particularly the purpose of the Springboard - that Navigation link will be live soon) and host a manageable changelog
  • Added the Downloadable / Printable Materials section, featuring the Controls Template that I created a few days ago and the Boilerplate templates that I created today
  • Brought over the existing Android Studio tutorials from the old staging site
  • Wrote the two tutorials for OpMode and LinearOpMode - I wrote LinearOpMode first then copied it and modified it for OpMode. It actually only took an hour or so.
  • Added some more features to global.css:
    • Standard font size options - class="small" now gives a font-size of 10pt and class="xsmall" gives a font-size of 8pt
    • Inline code tags - setting the class of a code tag to "inline" will render it with display: inline; rather than display: block; (There's two examples of that!)
    • Code colorization classes:
      • class="keyword" renders bold and blue
      • class="annotation" renders goldenrod
      • class="comment" renders gray
    • Improved comments

None of that actually involves the Springboard, despite that being the central focus of the site. I wanted to clear up everything that I had spun off before adding more stuff to it, and I did everything today, so perhaps tomorrow I can continue rounding out the Springboard.
