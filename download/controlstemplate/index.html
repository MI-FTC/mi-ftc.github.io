How to use this image:

Drag the controller elements to their locations on your controllers. (I left it up to the user to let you show both xBox and DualShock-style controllers.) Then mark up the image however you wish. I draw lines between each button and a textbox with a 1-3 word description of its function, but you could also use pictures or some other representation.


If these links don't work in your browser, the alternative is to save the file yourself. In the mainstream browsers at least, right-click on the link and select "Save Link As" from the context menu.

Available File Types:

  • Recommended: .odg - OpenDocument Format Drawing
    • Open in OpenOffice / (preferred) LibreOffice Draw or another drawing program that supports ODF to easily drag individual elements around.
    • If you cannot use this file type, choose your favorite from the list below, select each element, and drag the selection to the appropriate spot on the controller.
  • .bmp - Bitmap
  • .gif - Graphics Interchange Format
  • .jpg - Joint Photographic [Experts] Group
  • .png - Portable Network Graphic
  • .svg - Scalable Vector Graphics (shown in Preview)
  • .tiff - Tagged Image File Format
